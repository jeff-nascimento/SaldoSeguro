# Saldo Seguro

A financial management web application designed to help users track income, expenses, and category-based spending with a focus on performance, clean architecture, and user experience.

Um aplicativo web de gestão financeira desenvolvido para ajudar usuários a controlar ganhos, gastos e despesas por categoria, com foco em performance, arquitetura limpa e experiência do usuário.

---

## Purpose

This project was built to go beyond basic CRUD operations, focusing on real-world front-end practices such as separation of concerns, efficient DOM manipulation, and scalable architecture.

Este projeto foi desenvolvido para ir além de operações CRUD básicas, com foco em práticas reais de front-end como separação de responsabilidades, manipulação eficiente do DOM e arquitetura escalável.

---

## Features

- Add, edit, and remove transactions

- Adicionar, editar e remover transações

- Real-time currency input formatting

- Formatação de moeda em tempo real

- Dynamic form behavior based on transaction type

- Comportamento dinâmico do formulário baseado no tipo de transação

- Filter transactions by date and category

- Filtrar transações por data e categoria

- Interactive table with sorting and visual state feedback

- Tabela interativa com ordenação e feedback visual de estado

- Financial summary (income, expenses, balance)

- Resumo financeiro (ganhos, gastos, saldo)

- Interactive dashboard with charts

- Dashboard interativo com gráficos

- Visual feedback for user actions (highlight updates)

- Feedback visual para ações do usuário (destaque de atualizações)

- Persistent data using LocalStorage

- Persistência de dados utilizando LocalStorage

---

## Architecture Highlights

This project follows structured and scalable front-end architecture principles:

Este projeto segue princípios estruturados e escaláveis de arquitetura front-end:

- Clear separation of concerns (Models, Views, Services, Controllers)

- Separação clara de responsabilidades (Models, Views, Services, Controllers)

- Views responsible for DOM manipulation and UI state

- Views responsáveis pela manipulação do DOM e estado da interface

- Services encapsulating business logic and calculations

- Services encapsulando regras de negócio e cálculos

- Controllers orchestrating application flow

- Controllers orquestrando o fluxo da aplicação

- Event delegation for scalable and performant DOM interactions

- Uso de event delegation para interações de DOM mais escaláveis e performáticas

- Consistent data normalization and formatting strategies

- Estratégias consistentes de normalização e formatação de dados

- Avoidance of tightly coupled components

- Evita componentes fortemente acoplados
