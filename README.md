# Saldo Seguro

A financial management web application designed to help users track income, expenses, and category-based spending in a simple and organized way.

Um aplicativo web de gestão financeira desenvolvido para ajudar usuários a controlar ganhos, gastos e despesas por categoria de forma simples e organizada.

---

## Purpose

This project was built to practice TypeScript, modular architecture, and separation of responsibilities, focusing on building a scalable and maintainable front-end application.

Este projeto foi desenvolvido para praticar TypeScript, arquitetura modular e separação de responsabilidades, com foco em criar uma aplicação front-end escalável e de fácil manutenção.

---

## Features

- Add, edit, and remove transactions
- Adicionar, editar e remover transações

- Categorize income and expenses
- Categorizar ganhos e gastos

- Filter transactions by date and category
- Filtrar transações por data e categoria

- Financial summary (income, expenses, balance)
- Resumo financeiro (ganhos, gastos, saldo)

- Interactive dashboard with charts
- Dashboard interativo com gráficos

- Persistent data using LocalStorage
- Persistência de dados com LocalStorage

---

## Architecture Highlights

This project follows good development practices:

Este projeto segue boas práticas de desenvolvimento:

- Separation of concerns (Models, Views, Services, Controllers)
- Separação de responsabilidades (Models, Views, Services, Controllers)

- Avoidance of God Classes
- Evita classes gigantes (God Class)

- Reusable components and views
- Componentes e views reutilizáveis

- Clear and maintainable structure
- Estrutura clara e fácil de manter
